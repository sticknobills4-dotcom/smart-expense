# **App Name**: SmartExpense

## Core Features:

- Secure User Authentication: Enable user registration and login using email/password and Google accounts, integrated with Firebase Authentication.
- Personal Dashboard: Display an at-a-glance view of total balance, recent transactions, and a summary of income vs. expenses, retrieving data from Firestore.
- Transaction Management: Allow users to easily add, edit, and delete detailed income and expense transactions, assigning them to categories and accounts in Firestore.
- Multi-Account Tracking: Manage various financial accounts (e.g., Cash, Bank, Savings) and support fund transfers between them, storing account data in Firestore.
- Budget Planner: Create and track monthly budgets for custom categories, visualizing spending progress based on Firestore data.
- Expense Visualization: Provide basic charts to display spending breakdown by category and monthly spending trends using aggregated transaction data.
- Smart Spending Assistant: An AI tool that analyzes spending habits to offer personalized insights, detect potential overspending, and suggest budget adjustments based on historical transaction data.

## Style Guidelines:

- The chosen light color scheme is anchored in the principles of clarity and financial growth. A primary color is a clean, trustworthy blue (#2989D1), evoking a sense of reliability and modernity crucial for financial management.
- The background color is a heavily desaturated variant of the primary hue, providing a crisp, neutral canvas (#F0F3F5) that keeps data legible and prevents eye strain in a light theme.
- An accent color is a vibrant aqua (#2EDEB3), providing clear visual emphasis for interactive elements and positive actions, while creating dynamic contrast with the primary blue.
- The single font 'Inter' (sans-serif) is recommended for all text elements (headlines and body). Its modern, objective, and highly readable design ensures clarity for displaying financial data and maintains a sleek, professional aesthetic across the application.
- Use a set of modern, minimalist outline icons for clear visual cues, aligning with the app's clean and functional design aesthetic.
- Implement a clean, card-based dashboard layout that efficiently organizes financial information, optimized for mobile responsiveness with a prominent bottom navigation for intuitive access to main features.
- Subtle, fluid animations and transitions should be used sparingly for page transitions, data updates, and interactive elements to provide a smooth and engaging user experience without distracting from financial data.
